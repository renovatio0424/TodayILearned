## round(), floor(), ceil() [(Develop88)](https://developer88.tistory.com/97)

### 반올림

~~~java
double num = 0.5324
long convertedNum = Math.round(num)
// 1
~~~

### 내림

~~~java
double num = 0.5324
double convertedNum = Math.floor(num)
//0.0
~~~

### 올림

~~~java
double num = 0.5324
double covertedNum = Math.ceil(num)
//1.0
~~~
